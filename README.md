# 📧 Отправка показаний счетчика

Простая веб-страничка для быстрого составления письма для отправки показаний электросчетчика через установленное email приложение на смартфоне.

## 🎯 Возможности

- ⚡ **Быстро** — три касания и письмо готово
- 📱 **Мобильное** — оптимизировано для смартфонов
- 💾 **Локальное хранилище** — данные сохраняются только на вашем телефоне
- 🔒 **Безопасно** — никаких серверов, никаких облаков
- 🎨 **Интуитивный интерфейс** — минимум кнопок, максимум удобства

## 🚀 Как использовать

### Первый запуск

1. Откройте приложение
2. Заполните данные в окне настроек:
   - **Email получателя** — куда отправлять показания
   - **Тема письма** — название письма (по умолчанию: "Показания электросчетчика")
   - **Шаблон текста** — в шаблон можно добавить:
     - `[ТЕКУЩИЕ]` — текущие показания счетчика
     - `[ПРЕДЫДУЩИЕ]` — предыдущие показания
     - `[РАСХОД]` — расход за период (автоматический расчет)
3. Нажмите **СОХРАНИТЬ НАСТРОЙКИ**

### Ежемесячная отправка показаний

1. Откройте приложение
2. Введите **текущие показания** счетчика
3. Введите **предыдущие показания** (из прошлого месяца)
4. Нажмите **ДАЛЕЕ (ОТПРАВИТЬ)**
5. Откроется ваш почтовый клиент с готовым письмом
6. Опционально: прикрепите фото счетчика 📎
7. Нажмите отправить

### Изменение настроек

Нажмите на кнопку **⚙️** (шестеренка) в правом верхнем углу, чтобы отредактировать email, тему или шаблон письма.

## 📋 Пример шаблона письма

```
Здравствуйте!

Передаю показания счетчика электроэнергии:
- Текущие показания: [ТЕКУЩИЕ]
- Предыдущие показания: [ПРЕДЫДУЩИЕ]
- Расход за месяц: [РАСХОД] кВт.

Лицевой счет №123456789.
С уважением.
```

## 🔐 Безопасность

- ✅ Данные хранятся **локально** на вашем устройстве (localStorage браузера)
- ✅ **Не отправляются** на какой-либо сервер
- ✅ **Не собирается** никакая личная информация
- ✅ Защита от XSS-атак при вводе HTML-кода в шаблон

## 💻 Технические требования

- Современный браузер на смартфоне (Chrome, Safari, Firefox, Edge)
- Настроенный почтовый клиент по умолчанию
- Доступ в интернет (для отправки письма)

## 📦 Установка

Просто откройте `index.html` в браузере:

```bash
# Локально
file:///путь/до/index.html

# Или разместите на веб-сервере
https://your-domain.com/index.html
```

## 🎨 Что внутри

- Чистый **HTML5** + **CSS3** + **Vanilla JavaScript**
- Без зависимостей и библиотек
- Размер файла: ~10 KB
- Работает полностью **offline** после первой загрузки

## ⚠️ Важно

- Приложение использует **mailto:** для открытия почты, поэтому нужен настроенный почтовый клиент
- Данные хранятся локально и **удаляются при очистке кэша браузера**
- В приватном режиме браузера localStorage может быть недоступен
